## 1.2 Создай хук hooks/useProgress.ts

```ts
import { useEffect, useState } from 'react'
import { createClient } from '@/lib/supabase-client'
import * as progress from '@/lib/progress'

export function useProgress() {
  const [userId, setUserId] = useState<string|null>(null)
  const [profile, setProfile] = useState<any>(null)
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    const supabase = createClient()
    supabase.auth.getSession().then(({ data }) => {
      const uid = data.session?.user?.id ?? null
      setUserId(uid)
      if (uid) {
        progress.getUserProfile(uid).then(p => setProfile(p))
      }
      setLoading(false)
    })
  }, [])

  return { userId, profile, loading,
    completeLesson: (lessonId: string, score: number) => userId ? progress.completeLesson(userId, lessonId, score) : null,
    updateVocabWord: (wordId: string, box: number, days: number) => userId ? progress.updateVocabWord(userId, wordId, box, days) : null,
    logActivity: (mins: number, xp: number) => userId ? progress.logActivity(userId, mins, xp) : null,
    setLevel: (level: string, score: number) => userId ? progress.setUserLevel(userId, level, score) : null,
    refetchProfile: () => userId ? progress.getUserProfile(userId).then(setProfile) : null
  }
}
```

### MODULE: Academic Writing (15 уроков, B2-C1)

Module id prefix: aw-

Block 1 "Основы академического письма" (aw-1-1 по aw-1-5):
aw-1-1 "Академический стиль" (B2)
Theory: Academic writing characteristics: formal vocabulary, objective tone, hedging language, impersonal style, evidence-based arguments. Avoid: contractions (don't→do not), colloquial words (lots of→a significant number of), first person opinion (I think→It can be argued). Use: passive voice, nominalisation, formal connectors.
Exercises: 5 (transform informal→formal sentence)
Quiz: 5

aw-1-2 "Структура эссе" (B2)
Theory: Introduction: hook→background→thesis statement. Body paragraphs: topic sentence→evidence→analysis→link. Conclusion: restate thesis (not copy)→summary of points→final thought. Thesis statement must be debatable and specific. Example thesis: "Social media has fundamentally altered political discourse by creating echo chambers and enabling misinformation."
Exercises: 5 put paragraph elements in order / identify thesis
Quiz: 5

aw-1-3 "Аргументативное эссе" (B2)
Theory: Argumentative essay takes ONE position and defends it. Structure: Introduction with clear thesis→Point 1 (strongest)→Point 2→Counter-argument + refutation→Conclusion. Language: claim (It is evident that...), evidence (According to research...), analysis (This demonstrates that...), concession (Although critics argue...), refutation (Nevertheless, the evidence suggests...).
Exercises: 5
Quiz: 5

aw-1-4 "Дискуссионное эссе" (B2)
Theory: Discursive essay presents BOTH sides equally, then gives balanced conclusion. Structure: Introduction→Arguments for→Arguments against→Conclusion with your view. Difference from argumentative: you don't defend one side throughout. Language: On the one hand... / On the other hand... / Advocates of X argue that... / Opponents contend that...
Exercises: 5
Quiz: 5

aw-1-5 "Академическая лексика" (B2)
Theory: AWL (Academic Word List) — the 570 most common academic words. Categories: analyse, assess, evaluate, examine (critical thinking), indicate, demonstrate, suggest (evidence), significant, considerable, substantial (degree), whereas, although, despite (contrast), furthermore, moreover, consequently (connectors).
Exercises: 5 fill_blank с AWL словами
Quiz: 5

Block 2 "IELTS Academic Writing" (aw-2-6 по aw-2-10):
aw-2-6 "IELTS Task 1: Описание графиков" (B2)
Theory: Task 1 Academic = describe a visual (graph, chart, table, diagram). Must include: overview paragraph (main trend, not details), specific data with figures. 150+ words, 20 minutes. Structure: Introduction (paraphrase prompt)→Overview (2 main features)→Details paragraph 1→Details paragraph 2. Key phrases: The graph shows... / Overall, it is clear that... / ... stood at / rose to / fell from... to... / The most striking feature is...
Exercises: 5 practice describing mini-charts with sentence prompts
Quiz: 5

aw-2-7 "IELTS Task 1: типы заданий" (B2)
Theory: Types: Line graph (trends over time)→Bar chart (comparisons)→Pie chart (proportions)→Table (data comparison)→Process diagram (stages/how something works)→Map (changes to location). For process/map: describe stages in order. NO opinion in Task 1. Key language for each type.
Exercises: 5 identify type + choose correct language
Quiz: 5

aw-2-8 "IELTS Task 2: типы эссе" (B2/C1)
Theory: 5 types: 1) Opinion (Do you agree/disagree?) 2) Discussion (Discuss both views) 3) Problem-Solution 4) Two-part question 5) Advantages/Disadvantages. Must identify TYPE correctly — affects structure. 250+ words, 40 min. Assessed on: Task Response, Coherence & Cohesion, Lexical Resource, Grammatical Range.
Exercises: 5 identify essay type from question
Quiz: 5

aw-2-9 "IELTS Task 2: Band 7+ язык" (C1)
Theory: Lexical resource for high band: collocations (address an issue, tackle a problem, have a profound impact), precise vocabulary (alleviate poverty, exacerbate inequality, foster innovation), hedging (tends to, is likely to, may well). Grammatical range: complex sentences, conditionals, relative clauses, passives all in ONE essay.
Exercises: 5 improve sentences to B7+ level
Quiz: 5

aw-2-10 "IELTS Writing полный тест" (C1)
Theory: Strategy: Task 2 first (more marks)→Task 1. Planning: 5 min per task. Checking: grammar, spelling, word count. Common mistakes: copying question words (paraphrase!), not giving examples, going off topic, too informal.
Exercises: 2 полных Task 1 + Task 2 эссе с checklist
Quiz: 5 вопросов про IELTS правила

Block 3 "TOEFL Writing" (aw-3-11 по aw-3-15):
aw-3-11 "TOEFL: структура теста" (B2)
Theory: TOEFL iBT: Reading (54-72 min, academic passages)→Listening (41-57 min, lectures+conversations)→Speaking (17 min, 4 tasks)→Writing (50 min, 2 tasks). Writing Task 1 = Integrated (read passage 3 min → listen lecture → write 150-225 words how lecture relates to reading). Writing Task 2 = Independent (write opinion essay 300+ words in 30 min).
Exercises: 5 вопросов на структуру TOEFL
Quiz: 5

aw-3-12 "TOEFL Integrated Writing" (B2/C1)
Theory: Integrated task structure: Introduction (The reading argues X. However, the lecture challenges this by Y)→Point 1 (The reading states... However, the professor argues...)→Point 2→Point 3. KEY: always relate lecture BACK to reading. Do NOT include your opinion. Cast Doubt language: The professor challenges the claim that... / This undermines the reading's position that... / The lecture contradicts the reading by arguing...
Exercises: 5 practice matching reading points to lecture challenges
Quiz: 5

aw-3-13 "TOEFL Independent Writing" (B2/C1)
Theory: Independent task: Do you agree/disagree? OR Describe/explain OR Compare. 5-paragraph structure: Introduction (hook + clear thesis)→Body 1 (strongest point + example)→Body 2 (second point + example)→Body 3 (optional, or counter-argument)→Conclusion (restate, don't copy). Target: 400-500 words for 30/30. Clear position from the start.
Exercises: 5 thesis statement practice
Quiz: 5

aw-3-14 "TOEFL Speaking" (B2/C1)
Theory: 4 Speaking tasks. Task 1: Independent — express preference (45 sec to prepare, 60 sec to speak). Tasks 2-4: Integrated — read + listen + speak. Template Task 1: "I prefer X because of two reasons. First... Second... For these reasons, I believe X." Pronunciation tips: link words, stress key words, avoid "um/uh" — say "well, let me think".
Exercises: 5 complete spoken response templates
Quiz: 5

aw-3-15 "TOEFL vs IELTS: сравнение" (C1)
Theory: TOEFL: American English, computer-based, academic settings, 0-120 score. IELTS: British/International English, paper or computer, academic OR general training, 0-9 band. TOEFL preferred: USA/Canada universities. IELTS preferred: UK/Australia/Canada immigration. Key differences in writing task types, speaking format, reading passage style.
Exercises: 5 вопросов на различия
Quiz: 5
