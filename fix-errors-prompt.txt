## ШАГ 0: Сначала запусти проверку

```bash
npx tsc --noEmit 2>&1 | grep "^app/" | grep -oP "app/[^(]+" | sort -u
```

Ты увидишь список файлов с ошибками. Исправь каждый.

---

## ШАГ 1: Для каждого файла с ошибкой — алгоритм исправления

Для каждого файла из списка:

1. Прочитай файл полностью (`Read`)
2. Найди ошибку TypeScript: `npx tsc --noEmit 2>&1 | grep "filename"`
3. Посмотри на строку с ошибкой — что там
4. Определи что не закрыто или задублировано
5. Исправь минимально (`Edit`)
6. Проверь: `npx tsc --noEmit 2>&1 | grep "filename"` — ошибок быть не должно
