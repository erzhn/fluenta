Логика shuffle (добавь в файл):
```ts
function shuffleArray<T>(arr: T[]): T[] {
  const a = [...arr]
  for (let i = a.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [a[i], a[j]] = [a[j], a[i]]
  }
  return a
}
```

При старте теста: `const questions = shuffleArray(QUESTION_POOL).slice(0, 20)`
Подсчёт уровня: считай правильные по уровням — побеждает тот уровень, где больше всего правильных плюс все предыдущие.

Упрощённая логика: общий score 0-20:

## ЗАДАЧА 2: АУДИРОВАНИЕ — Groq генерирует новые тексты

### 2a. Создай app/api/ai/generate-listening/route.ts

```ts
import { NextResponse } from 'next/server'

export async function POST(req: Request) {
  const { level } = await req.json()
  
  const prompt = `Generate an English listening exercise for level ${level}.
Return ONLY valid JSON (no markdown) with this exact structure:
{
  "title": "short title in English",
  "text": "the text/dialogue to listen to (3-6 sentences for A1-A2, 6-10 for B1-B2, 8-12 for C1)",
  "questions": [
    { "question": "comprehension question", "options": ["a","b","c","d"], "answer": "correct option" },
    { "question": "comprehension question", "options": ["a","b","c","d"], "answer": "correct option" },
    { "question": "comprehension question", "options": ["a","b","c","d"], "answer": "correct option" }
  ]
}

Rules:
- A1: very simple dialogue (café, shop, meeting someone). Short sentences. Common vocabulary.
- A2: everyday situation (travel, phone call, making plans). Some past tense.
- B1: slightly complex topic (advice, news story, describing a problem). Mix of tenses.
- B2: article or lecture excerpt (technology, society, environment). Academic-ish vocabulary.
- C1: complex discussion or lecture (philosophy, economics, science). Sophisticated language.
- Make questions genuinely test comprehension, not just copy sentences.
- All 3 wrong options should be plausible.`

  const resp = await fetch('https://api.groq.com/openai/v1/chat/completions', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${process.env.GROQ_API_KEY}` },
    body: JSON.stringify({
      model: 'llama-3.1-8b-instant',
      messages: [{ role: 'user', content: prompt }],
      max_tokens: 800,
      temperature: 0.85,
    })
  })
  
  const data = await resp.json()
  const raw = data.choices?.[0]?.message?.content ?? '{}'
  
  try {
    const match = raw.match(/\{[\s\S]*\}/)
    const result = JSON.parse(match ? match[0] : raw)
    return NextResponse.json(result)
  } catch {
    return NextResponse.json({ error: 'Parse error' }, { status: 500 })
  }
}
```
