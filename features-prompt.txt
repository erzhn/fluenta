### app/(dashboard)/vocabulary/page.tsx

SRS (Spaced Repetition System) с локальным хранилищем.

localStorage ключ 'fluenta_vocab_srs':
```ts
{
  [wordId: string]: {
    interval: number      // в днях (1, 3, 7, 14, 30)
    dueDate: string       // ISO date
    box: number           // 0-5 (Leitner boxes)
    seen: boolean
  }
}
```

Режимы страницы:
1. ОБЗОР (default):
   - Вверху: "Сегодня к повторению: X слов" + "Изучено всего: Y слов"
   - Фильтры по уроку (dropdown или tabs по блокам A1)
   - Grid карточек: слово + перевод, статус (новое/к повторению/выучено)
   - Кнопка "Учить сегодня (X)" → режим FLASHCARDS

2. FLASHCARDS:
   - Показывает только слова с dueDate <= сегодня ИЛИ новые (box=0)
   - Карточка: front = английское слово (большой шрифт)
   - Нажать/кликнуть → flip (CSS transform rotateY 180deg): back = перевод + пример
   - Кнопка 🔊 → озвучить через SpeechSynthesis
   - Внизу 3 кнопки: "❌ Не знаю" (box→0, interval=1) / "🤔 Помню" (box+1, interval*2) / "✅ Знаю!" (box+1, interval*2, max 30)
   - Прогресс: "5 из 12 слов"
   - После всех: "Готово! Следующее повторение: через X дней"

## ЗАДАЧА 3: СТРИК И ЕЖЕДНЕВНАЯ ЦЕЛЬ
Файл: components/StreakWidget.tsx (используй в dashboard)

localStorage ключ 'fluenta_streak':
```ts
{
  currentStreak: number
  longestStreak: number
  lastActiveDate: string // ISO date YYYY-MM-DD
  todayMinutes: number
  dailyGoalMinutes: number // default 10
  history: { [date: string]: number } // date → minutes
}
```

Логика:
- При каждом взаимодействии с уроком/словарём/тестом → вызывать trackActivity(minutes: number)
- trackActivity проверяет: если lastActiveDate < today → новый день. Если вчера → streak+1. Если нет → streak=1.
- Если todayMinutes >= dailyGoalMinutes → цель выполнена.

Компонент StreakWidget:
- Показывает: 🔥 {currentStreak} дней
- Прогресс бар: "Сегодня: X мин из 10"
- Если цель выполнена: зелёный бейдж "✅ Цель выполнена!"
- Последние 7 дней: маленькие кружки (зелёный = был активен, серый = нет)
- Настройка цели: клик на "10 мин" → popup сменить на 5/10/15/20/30
