Lesson a2-2-36: "Zero Conditional"
Theory: Zero Conditional = facts and general truths. If + Present Simple, Present Simple. The result is ALWAYS true. Can replace "if" with "when". If you heat water to 100°C, it boils.
Examples: If you eat too much, you get fat. / Water freezes if the temperature is below zero. / If it rains, I get wet. / When you don't sleep, you feel tired. / If you exercise regularly, you feel better. / Plants die if they don't get water.

Lesson a2-2-37: "First Conditional"
Theory: First Conditional = real and possible future situations. If + Present Simple, WILL + base verb. Possible situation in the future with a likely result. Can also use: might, can, should in the result clause.
Examples: If it rains tomorrow, I will stay home. / If you study hard, you will pass the exam. / If she calls, I'll tell her. / I'll be angry if you're late. / If we leave now, we won't miss the train. / If you need help, you can call me.

Lesson a2-2-38: "Unless"
Theory: UNLESS = if not. Unless you study, you'll fail = If you don't study, you'll fail. Unless is only used in real/zero conditionals (not 2nd or 3rd). Always use UNLESS + positive verb.
Examples: I'll go unless it rains. / Unless you hurry, you'll be late. / She won't come unless you invite her. / Unless you eat breakfast, you'll be hungry. / He'll call unless he's busy. / I won't tell anyone unless you ask me to.

Lesson a2-3-42: "Must vs Have to"
Theory: MUST = internal obligation (you think it's important). HAVE TO = external obligation (a rule, law, or someone requires it). Mustn't = prohibition (don't do it!). Don't have to = not necessary (free choice).
Examples: I must call my mother — I haven't spoken to her in a week. / You have to wear a seatbelt — it's the law. / You mustn't smoke in here. / You don't have to come if you don't want to. / I must remember to buy milk. / Students have to hand in their work by Friday.

Examples: If I had more time, I would learn to paint. / If she were taller, she could be a model. / Would you move abroad if you could? / If I won the lottery, I'd buy a house. / What would you do if you lost your job? / If I were you, I would apologise.

Lesson b1-1-52: "Third Conditional"
Theory: Third Conditional = imaginary past (regrets about what didn't happen). If + Past Perfect, would/could/might + have + past participle. If I had studied harder, I would have passed. (But I didn't study, so I didn't pass.)
Examples: If I had studied harder, I would have passed the exam. / She would have caught the train if she had woken up earlier. / If they had told me, I would have helped. / I wouldn't have got lost if I had used a map. / If he had eaten breakfast, he wouldn't be hungry now. / Would you have come if you had known?

Lesson b1-1-53: "Mixed Conditionals intro"

Examples: If I had worked harder at school, I would have a better job now. / If she weren't so shy, she would have spoken to him. / If I had been born in England, I would speak perfect English. / He would have more money if he didn't spend so much. / If I had taken that job, I would be living in New York now. / She wouldn't be tired now if she had slept last night.

Lesson b1-1-54: "I wish / If only"

Examples: I wish I hadn't eaten so much. / If only I had studied harder! / She wishes she hadn't quit her job. / I wish I could undo what I said. / If only we had left earlier! / I wish you would listen to me.

Lesson b1-5-71: "Must/Can't for deduction"
Theory: MUST = I'm almost certain it's true (positive deduction). CAN'T = I'm almost certain it's not true (negative deduction). These are about present situations based on evidence. She must be tired — she's been working all day. He can't be at home — his car isn't there.
